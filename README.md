# Level-Locked-Picking
An SKSE plugin that adds skill level requirments to being able to lockpick containers.

## Requirements
* [CMake](https://cmake.org/)
* [vcpkg](https://github.com/microsoft/vcpkg)
   * Add an environment variable called "VCPKG_ROOT", with the value being the folder where you installed vcpkg.
* [Visual Studio Community 2022](https://visualstudio.microsoft.com/)
